<h1>Physics-Informed Federated Learning with Interpretability: A Framework for Privacy-Preserving Joint SoC and SoH Estimation in Electric Vehicle Fleets</h1>

</p>

<p align="justify">
This experiment is on the <b>Battery Management System (BMS)</b>, where the estimation of <b>SoC (State of Charge)</b> and <b>SoH (State of Health)</b> is done using the integration of <b>Deep Learning (DL)</b> and <b>Physics Informed Neural Networks (PINNs)</b>. This experiment also ensures privacy using the concept of <b>Federated Learning (FL)</b>.
</p>
